Create next season
2 buttons on this form. Create new season creates a row in the seasons table for the next year. Run this first
Second button imports the schedule from a .csv file that you must create and import into the resources/data folder.
That file is created from downloading the schedule from Here
This download should be brought into a spreadsheet from which you can create the csv file. Then button 2 can be run to update the nflgame table.
